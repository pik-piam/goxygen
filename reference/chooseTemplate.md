# chooseTemplate

Support function helping to choose the selected template

## Usage

``` r
chooseTemplate(style, templatefolder, ftype = NULL)
```

## Arguments

- style:

  visualization style to be used for the creation.

- templatefolder:

  Folder in which goxygen will search for template files in addition to
  the pre-installed ones.

- ftype:

  template file type / file ending, or NULL if looking for a whole
  directory
